       ├─ Listar e buscar produtos
       ├─ Gerar relatórios

OPÇÃO 1: Começar com dados de exemplo (RECOMENDADO)
─────────────────────────────────────────────────────────────────────────
1. Copie: dados_estoque_exemplo.json → dados_estoque.json
   (Use: copy dados_estoque_exemplo.json dados_estoque.json)

2. Execute: python estoque.py

3. Use a opção 5 para ver os 10 produtos de exemplo

4. Experimente as operações (entrada, saída, relatórios)


OPÇÃO 2: Começar do zero
─────────────────────────────────────────────────────────────────────────
1. Execute: python estoque.py

2. Escolha opção 1 e adicione seus primeiros produtos

3. Sistema cria dados_estoque.json automaticamente


OPÇÃO 3: Executar testes automáticos
─────────────────────────────────────────────────────────────────────────
1. Execute: python teste.py

2. Ver todas as funcionalidades funcionando

3. Gera arquivo: teste_estoque.json

║                           ESTRUTURA DE DADOS                              ║
╚═══════════════════════════════════════════════════════════════════════════╝

CADA PRODUTO CONTÉM:
─────────────────────────────────────────────────────────────────────────
  ID          → Identificador único (ex: PROD001)
  Nome        → Nome do produto
  Categoria   → Tipo/classificação
  Preço       → Valor unitário
  Quantidade  → Quantidade atual em estoque
  Mínimo      → Limite para alerta
  Localização → Onde fica no armazém/loja
  Data        → Quando foi adicionado


╔═══════════════════════════════════════════════════════════════════════════╗
║                         MENU DO SISTEMA (10 OPÇÕES)                       ║
╚═══════════════════════════════════════════════════════════════════════════╝

1️⃣  Adicionar Produto         ├─ Cadastra novo item
2️⃣  Remover Produto           ├─ Deleta um produto
3️⃣  Entrada de Estoque        ├─ Registra compra/recebimento
4️⃣  Saída de Estoque          ├─ Registra venda/saída
5️⃣  Listar Todos os Produtos  ├─ Mostra todos com detalhes
6️⃣  Buscar Produto            ├─ Procura por ID específico
7️⃣  Produtos com Estoque Baixo ├─ Alerta de reposição
8️⃣  Relatório de Valor Total  ├─ Mostra valor investido
9️⃣  Relatório por Categoria   ├─ Organiza por tipo
0️⃣  Sair                      └─ Encerra o programa
