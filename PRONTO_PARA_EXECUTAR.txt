❌ SE ALGO DER ERRADO
═════════════════════════════════════════════════════════════════════════════════

Erro: "MySQL not found"
───────────────────────────────────────
1. Instale MySQL: https://dev.mysql.com/downloads/mysql/
2. IMPORTANTE: Marque "Add MySQL to PATH" na instalação
3. Reinicie este setup

Erro: "Access denied" ao conectar
───────────────────────────────────────
1. Verifique arquivo .env
2. Verifique PASSWORD no setup
3. Teste: mysql -u estoque_user -p estoque_db
4. Senha padrão: 12345

Erro: "Porta 5000 já em uso"
───────────────────────────────────────
1. Edite app.py, linha com app.run()
2. Altere: port=5000 → port=8000
3. Acesse em: http://localhost:8000

Erro: "Module not found" (Flask, pymysql, etc)
───────────────────────────────────────────────
1. Execute: python instalar_dependencias.py
2. OU: pip install -r requirements.txt

MySQL não inicia/conecta
───────────────────────────────────────
Windows:
  • Abra Serviços (Services)
  • Procure por "MySQL"
  • Clique direito → Iniciar

Linux:
  • sudo service mysql start
  • OU: sudo systemctl start mysql


🔄 PARA USAR EM MÚLTIPLAS MÁQUINAS
═════════════════════════════════════════════════════════════════════════════════

Cenário: Uma máquina com MySQL central, outras apenas clients

MÁQUINA 1 (Servidor com MySQL):
───────────────────────────────────────
1. Execute setup_completo.py (como descrito acima)
2. MySQL roda em: 192.168.1.100:3306 (exemplo IP)
3. Inicie servidor: python app.py
4. Anotar IP: 192.168.1.100


MÁQUINAS 2, 3, 4... (Clientes):
───────────────────────────────────────
1. Instale Python (as dependências)
   pip install -r requirements.txt

2. Edite arquivo .env:
   DATABASE_URL=mysql+pymysql://estoque_user:12345@192.168.1.100:3306/estoque_db
   (Altere 192.168.1.100 pelo IP real da máquina com MySQL)

3. Inicie servidor:
   python app.py

4. Acesse de qualquer navegador:
   http://192.168.1.100:5000


🔐 SEGURANÇA
═════════════════════════════════════════════════════════════════════════════════

ANTES DE COLOCAR EM PRODUÇÃO:

⚠️ Altere .env:
   SECRET_KEY=altere-isto-para-algo-realmente-secreto
   (Use algo como: openssl rand -hex 32)

⚠️ Altere senha padrão do admin:
   • Login em: http://localhost:5000
   • Usuário: admin
   • Senha atual: admin
   • Edite seu perfil (aba Admin)

⚠️ Altere senha do MySQL:
   mysql -u root -p
   ALTER USER 'estoque_user'@'%' IDENTIFIED BY 'nova_senha_segura';
   (E atualize .env com a nova senha)

⚠️ Configure firewall:
   • Abra porta 5000 (Flask)
   • Abra porta 3306 (MySQL) apenas pra IPs confiáveis
