## Instalação

### Instalar todas as dependências:
```bash
pip install -r requirements.txt
```

ou

```bash
python -m pip install -r requirements.txt
```

### Como Executar

```bash
python app.py
```

Para o sistema de estoque padrão:
```bash
python estoque.py
```

Para testes:
```bash
python teste.py
```

## Troubleshooting

### "Python não é reconhecido como comando"
- Reinstale Python e marque "Add Python to PATH"
- Ou use: `python3 estoque.py`

### "ModuleNotFoundError"
- Geralmente não ocorre (usa bibliotecas padrão)
- Verifique se está usando Python 3.6+

### "Arquivo não encontrado"
- Execute `python estoque.py` dentro da pasta ESTOQUE
- Ou especifique o caminho completo

### Dados não são salvos
- Verifique permissões de escrita na pasta
- Certifique-se que está no diretório correto
