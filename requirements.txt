# Sistema de Gestão de Estoque - Requisitos

## Requisitos de Sistema

### Python
- Python 3.6 ou superior
- Verifique sua versão:
  ```
  python --version
  ```

### Dependências
- **Nenhuma dependência externa!**
- O sistema usa apenas bibliotecas padrão do Python:
  - `json` - para armazenar dados
  - `os` - para operações do sistema
  - `datetime` - para datas e horas
  - `typing` - para type hints

## Instalação

### 1. Baixar/Copiar os Arquivos

Certifique-se de ter estes arquivos:
- `estoque.py` - Arquivo principal
- `teste.py` - Script de testes
- `dados_estoque_exemplo.json` - Dados de exemplo
- `README.md` - Documentação
- `GUIA_RAPIDO.txt` - Guia de operações

### 2. Preparar Dados Iniciais (Opcional)

Se quer começar com dados de exemplo:
```bash
# Copie o arquivo de exemplo
copy dados_estoque_exemplo.json dados_estoque.json
```

## Como Executar

### Modo Interativo (Menu)
```bash
python estoque.py
```

### Modo Teste
```bash
python teste.py
```

## Troubleshooting

### "Python não é reconhecido como comando"
- Reinstale Python e marque "Add Python to PATH"
- Ou use: `python3 estoque.py`

### "ModuleNotFoundError"
- Geralmente não ocorre (usa bibliotecas padrão)
- Verifique se está usando Python 3.6+

### "Arquivo não encontrado"
- Execute `python estoque.py` dentro da pasta ESTOQUE
- Ou especifique o caminho completo

### Dados não são salvos
- Verifique permissões de escrita na pasta
- Certifique-se que está no diretório correto
