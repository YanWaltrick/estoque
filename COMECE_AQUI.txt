🚀 INICIAR EM 3 PASSOS
═══════════════════════════════════════════════════════════════════════════════

OPÇÃO 1: WINDOWS (Mais Fácil)
────────────────────────────

1. Duplo clique em: iniciar_frontend.bat
2. Aguarde o servidor iniciar (~5 segundos)
3. Abra no navegador: http://localhost:5000

Pronto! 🎯


OPÇÃO 2: QUALQUER SISTEMA (Python)
──────────────────────────────────

1. Abra terminal/CMD no diretório do projeto

2. Instale dependências:
   $ pip install -r requirements.txt

3. Inicie o servidor:
   $ python app.py

4. Abra no navegador: http://localhost:5000

Pronto! 🎯


OPÇÃO 3: VALIDAR INSTALAÇÃO ANTES
─────────────────────────────────

1. Execute o teste:
   $ python teste_frontend.py

2. Se tudo passar com ✓:
   $ python app.py

3. Abra: http://localhost:5000

═══════════════════════════════════════════════════════════════════════════════
❓ PERGUNTAS FREQUENTES
═══════════════════════════════════════════════════════════════════════════════

P: Python não apareceu como instalado
R: Reinstale Python e marque "Add Python to PATH"

P: Erro ao instalar dependências
R: Execute: pip install --upgrade pip
   Depois: pip install -r requirements.txt

P: Porta 5000 em uso
R: Edite app.py, procure "port=5000" e mude para "port=5001"

P: Dados não aparecem
R: Copie dados_estoque_exemplo.json para dados_estoque.json

P: Como parar o servidor?
R: Pressione CTRL+C no terminal

P: Funciona no celular?
R: Sim! Acesse http://SEU_IP:5000 (ex: http://192.168.1.5:5000)
