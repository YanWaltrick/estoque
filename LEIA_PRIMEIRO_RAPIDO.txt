┌─────────────────────────────────────────────────────────────────────────────┐
│                     ⚠️  SE ENCONTRAR ERROS (Troubleshooting)               │
└─────────────────────────────────────────────────────────────────────────────┘

"python not found"
→ Instale Python com "Add to PATH"

"mysql not found"
→ Instale MySQL com "Add to PATH"

"Access denied" (MySQL)
→ Verifique arquivo .env (senha padrão: 12345)

"ModuleNotFoundError" (Flask, sqlalchemy, etc)
→ Execute: python instalar_dependencias.py

"Porta 5000 já em uso"
→ Edite app.py, altere port=5000 para port=8000
